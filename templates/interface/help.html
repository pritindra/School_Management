The basic interface is the screen you see after logging in. It has tabs like categories,dashboard,reports,etc. which can be utilized by clicking on them. The dashboard section is where the graphs, charts and other data is visualized for better maintenance of the system.
The profile section is where you can view the complete description of the user logged in , including all the reports and information of the user.
Teh categories section is the part where it contains all the information of the system records maintained. It has detail view of each and every record based on their categories.
